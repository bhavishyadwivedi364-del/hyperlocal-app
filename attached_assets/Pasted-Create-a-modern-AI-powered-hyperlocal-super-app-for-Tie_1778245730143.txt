The app should work for both customers and sellers in a single platform with role-based login.

Tech Stack:
- React + Next.js frontend
- Node.js + Express backend
- Firebase Authentication with OTP login
- Firestore database
- Tailwind CSS
- Responsive for mobile and desktop
- Dark mode and light mode
- Multi-language support (English and Hindi initially)

- OTP login with phone number
- Home page with categories

  - Furniture
  - Taxi booking
  - Vegetables

- Product listing with images, prices, ratings
- Add to cart

- Cash on delivery option
- Order tracking

- Analytics dashboard
